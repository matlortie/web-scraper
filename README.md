# README

This is a basic web scraper where you can check for specific html tags and that sends email when the rake task is a success (could be run in cron job)

* Ruby version => 3.3.6

* Rails version => 7.2.2

* System dependencies
  - postgresql

* Database creation

`rails db:create`

* Database initialization

`rails db:migrate`

* How to run the test suite

`rails test`
